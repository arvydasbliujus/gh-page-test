Top secret!

Užduotis:

  1. Terminale su komanda (cd) nueiti į Desktop direktoriją
  2. Desktop'e su komanda (mkdir) sukurti aplanką/folderį pavadinimu mano_aplankas
  3. Su komanda (cd) nueiti į savo sukurtą aplanką/folderį
  4. Su komanda (code . ) atidaryti savo aplanką/folderį su Visual Studio Code programa. (Jeigu neatidaro kviesti mokytoją)
  5. Visual Studio Code programoje sukurti file'ą pavadinimu index.html ir tame file sukurti HTML skiletą. Išsaugoti file'ą
  6. Index.html file'o tag'e <title> </title> pakeisti pavadinimą į Mano aplankas.
  7. Index.html file'o tag'e <body> </body> sukurti tag'ą <h1> </h1> ir jame parašyti savo vardą ir žodį aplankas pvz.: Lino aplankas ir išsaugoti
  8. Visual Studio Code programoje sukurti aplanką/folderį pavadinimu css
  9. Visual Studio Code programoje css aplanke/folderyje sukurti file'ą pavadinimu main.css
  10. index.html file sujungti index.html ir main.css file'us hint:(link:css)
  11. main.css file parašyti komandą, kuri pakeistų tag'o html (foną) background'ą į raudoną
  12. main.css file parašyti komandą, kuri pakeistų tag'o h1 (spalvą) color į baltą
  13. Parodyti savo rezultatą naršyklėje


Mano gavosi toks :)
